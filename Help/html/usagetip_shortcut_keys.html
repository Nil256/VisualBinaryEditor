ショートカットキー

バイナリファイル

  • Ctrl + N: 新規作成

    現在の編集を破棄し、新しいバイナリファイルの作成を始めます。

  • Ctrl + O: 開く

    バイナリオーダーをもとに、バイナリファイルを開きます。

  • Ctrl + R: 再読み込み

    編集中のバイナリファイルを、バイナリオーダーをもとに開き直します。
    編集中のバイナリファイルが存在しない場合は何もしません。

  • Ctrl + S: 保存

    編集中のバイナリファイルを保存します。
    編集中のバイナリファイルが新規作成の場合、名前を付けて保存と同じ挙動をします。

  • Ctrl + Shift + S: 名前を付けて保存

    編集中のバイナリファイルを、編集中のものとは別に新しく保存します。

バイナリエントリ

  • Delete: バイナリエントリの削除

    選択中のバイナリエントリを削除します。

  • Shift + Delete: バイナリエントリの全削除

    バイナリオーダーを空にします。
