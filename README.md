# Visual Binary Editor

[System.IO.BinaryReader](https://learn.microsoft.com/ja-jp/dotnet/api/system.io.binaryreader?view=net-8.0), [System.IO.BinaryWriter](https://learn.microsoft.com/ja-jp/dotnet/api/system.io.binarywriter?view=net-8.0)によるバイナリファイルの読み込み、生成、編集を視覚的に行うことを目的としたソフトウェアです。
**このソフトウェアはまだアルファ版であり、現時点(v0.1.0-alpha)では、バイナリファイルの生成のみ実装されています。**

## 機能

バイナリファイルの生成ができます。
